# java-blackjack

블랙잭 미션 저장소

- [ ] 카드를 섞는 기능
- [ ] 카드를 분배하는 기능 
    - 플레이어
      - [ ] 입력에 따라서 카드를 받을지말지 결정하는 기능
    - 딜러
      - [ ] 숫자가 16이하면 카드를 받는 기능
- [ ] 게임의 결과를 반환하는 기능
  - [ ] 딜러가 17 이상일 때 승부를 반환하는 기능
